# 🎮 MiniGameHub - 多人联机小游戏平台

> 🚀 **一个让你和朋友随时随地开黑的实时多人游戏平台！**

基于 WebSocket 的实时多人联机游戏平台，支持语音聊天、断线重连、观战系统，带来沉浸式的在线游戏体验。

---

## 🌟 核心亮点

### 🎙️ 实时语音聊天
- **WebRTC 点对点音频传输** - 超低延迟，如同面对面交流
- **STUN/TURN 服务器自动切换** - 无论什么网络环境都能畅聊
- **一键开关语音** - 绿色呼吸灯状态提示，操作简单直观
- **智能静音** - 特定游戏模式自动禁用语音，保证公平性

### 🔄 断线重连系统
- **自动检测断线** - 毫秒级感知网络变化
- **无缝重连** - 1秒内自动恢复连接
- **游戏状态同步** - 重连后完整恢复棋盘/画布状态
- **语音自动恢复** - 重连后语音通话自动重建
- **断线玩家接手** - 观战者可接手断线玩家继续游戏

### 👁️ 实时观战系统
- **多人同时观战** - 不限人数围观精彩对局
- **实时状态同步** - 棋盘/画布变化毫秒级同步
- **观战人数显示** - 实时显示当前观战人数
- **无缝切换** - 从观战到接手，一键切换身份

## 🎯 精选游戏

### ♟️ 五子棋 (Gomoku)
> 经典策略对弈，考验智慧与耐心

**游戏特色：**
- 🎯 **15×15 专业棋盘** - Canvas 高清渲染，丝滑流畅
- ⚫ **双人实时对弈** - 黑白棋轮流落子，五连珠获胜
- 👁️ **观战模式** - 朋友对局时可以围观学习
- 🔄 **接手功能** - 断线玩家的空位可由观战者接手
- ↩️ **悔棋功能** - 对手同意后可撤销上一步
- 💬 **实时聊天** - 边下棋边聊天，社交娱乐两不误
- 🎵 **落子音效** - 清脆的落子声，增强游戏体验
- 📱 **双操作模式** - 快速模式（长按预览松手落子）/ 确认模式（点击选择再确认）
- 🎮 **再来一局** - 游戏结束后一键重开，无需重新建房

**胜利动画：**
- 五子连珠时触发华丽的胜利动画
- 闪光特效 + 棋子高亮，仪式感满满

| 6 | 1️⃣ 一笔封神 | 60秒 | 只能画一笔，一气呵成 |

| 9 | 📉 越来越细 | 60秒 | 笔刷自动变细 |

**🏆 评级系统：**

| 等级 | 条件 | 称号 |
|:---:|------|------|
| D | 默认 | 初识阶段 |
| C | 正确率 ≥ 40% | 普通朋友 |
| B | 正确率 ≥ 60%，连击 ≥ 3 | 默契搭档 |
| A | 正确率 ≥ 75%，连击 ≥ 5 | 心有灵犀 |
| S | 正确率 ≥ 85%，连击 ≥ 7 | 灵魂共鸣 |
| SS | 正确率 ≥ 90%，连击 ≥ 10 | 双人成神 |
| SSS | 正确率 ≥ 95%，连击 ≥ 15 | 无需语言 |

**♾️ 无限模式：**
- 通关18关后解锁
- 初始300秒，猜对+20秒并交换身份
- 每轮从所有关卡词库中随机抽取
- 50轮后解锁专属词库

**📚 海量词库：**
- **1368+ 个词语**，跨关卡去重
- 涵盖动物、水果、职业、地点、网络热词等
- 同一房间内绝不重复，保证新鲜感

### 🔢 数独 (Sudoku)
> 经典数字谜题，双人协作解谜

**游戏特色：**
- 🧩 **协作模式** - 两人分工合作，共同完成数独
- 🎚️ **三种难度** - 简单/中等/困难，满足不同水平
- 🔍 **智能提示** - 每局3次提示机会，关键时刻助你一臂之力
- ⏱️ **计时系统** - 记录完成时间，挑战自我极限
- ↩️ **撤销功能** - 输错了？一键撤销，轻松回退
- ✅ **实时验证** - 即时检查答案，错误一目了然

**难度说明：**
- 🟢 **简单** - 30-40个给定数字，新手友好
- 🟡 **中等** - 25-29个给定数字，进阶挑战
- 🔴 **困难** - 20-24个给定数字，高手对决

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0
- 现代浏览器（Chrome/Firefox/Safari/Edge）

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/wangmuerxiao/MiniGameHub.git
cd MiniGameHub

# 2. 安装后端依赖
cd server
npm install

# 3. 启动服务器
npm start

# 4. 打开浏览器访问
# http://localhost:3000
```

### 开发模式
```bash
# 自动重启，热更新
npm run dev
```

## 🌐 部署指南

### 本地部署
```bash
cd server
npm install
npm start
```

### PM2 生产部署（推荐）
```bash
# 安装 PM2
npm install -g pm2

# 启动服务
cd server
pm2 start server.js --name "minigamehub"

# 设置开机自启
pm2 startup
pm2 save

# 查看日志
pm2 logs minigamehub
```

### Docker 部署
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY server/package*.json ./server/
RUN cd server && npm install --production
COPY . .
EXPOSE 3000
CMD ["node", "server/server.js"]
```

```bash
# 构建并运行
docker build -t minigamehub .
docker run -d -p 3000:3000 --name minigamehub minigamehub
```

### Nginx 反向代理
```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 🎮 游戏玩法

### 五子棋快速上手
1. 🏠 进入游戏大厅，点击「五子棋」
2. 📋 创建房间，复制6位房间码
3. 📤 发送给朋友，等待加入
4. ⚫ 白棋先行，轮流落子
5. 🏆 五子连珠者获胜！

### 画猜快速上手
1. 🎨 进入游戏大厅，点击「默契画猜」
2. 📋 创建房间，邀请好友
3. ✅ 双方点击「准备」
4. 🖌️ 一人画，一人猜
5. 🎯 猜对得分，挑战更高关卡！

### 数独快速上手
1. 🔢 进入游戏大厅，点击「数独」
2. 🎚️ 选择难度级别
3. 👫 邀请搭档加入
4. 🧩 分工协作，共同解谜
5. ✅ 完成数独，记录时间！

## 💡 使用技巧

### 五子棋技巧
- 🎯 开局抢占天元（中心点）
- 🔍 注意对手的活三、冲四
- 💬 善用语音与对手交流

### 画猜技巧
- 🎨 先画轮廓，再填充细节
- 🗣️ 善用语音描述（除第13关）
- ⏰ 注意时间分配，别在一个细节上卡太久

### 数独技巧
- 🧩 先填确定性高的数字
- 👫 与搭档分工，避免重复劳动
- 💡 善用3次提示机会
