# 🎨 默契画猜

双人合作画猜闯关游戏。一人画，一人猜，18 个关卡各有独特机制。

## 快速开始

1. 进入游戏大厅，选择「默契画猜」
2. 创建房间，复制房间码发给朋友
3. 朋友输入房间码加入
4. 双方点击「准备」开始游戏

## 基本规则

- ✅ 可以画画
- ✅ 可以语音交流
- ❌ 不可以说出词语的任何字

## 无限模式

通关 18 关后解锁。初始 300 秒，猜对 +20 秒并交换身份。每轮从所有关卡词库中随机抽取，50 轮后解锁专属词库。

## 评级系统

根据正确率和最高连续猜中次数评定：

| 等级 | 条件 | 称号 |
|---|---|---|
| D | 默认 | 初识阶段 |
| C | 正确率 ≥ 40% | 普通朋友 |
| B | 正确率 ≥ 60%，连击 ≥ 3 | 默契搭档 |
| A | 正确率 ≥ 75%，连击 ≥ 5 | 心有灵犀 |
| S | 正确率 ≥ 85%，连击 ≥ 7 | 灵魂共鸣 |
| SS | 正确率 ≥ 90%，连击 ≥ 10 | 双人成神 |
| SSS | 正确率 ≥ 95%，连击 ≥ 15 | 无需语言 |

## 语音功能

- 使用 WebRTC 点对点音频传输
- STUN/TURN 服务器自动切换
- 开启语音后按钮变为 🟢 绿色呼吸灯
- 第 13 关强制关闭语音，关卡结束后恢复

## 技术实现

### 关卡机制

每个关卡通过 `mechanics` 对象配置：

```javascript
{
  oneStroke: true,     // 一笔封神
  inkFade: 3,          // 隐形墨水（秒）
  brushGrow: true,     // 越来越粗
  brushShrink: true,   // 越来越细
  autoClear: 10,       // 倒计时爆炸（秒）
  canvasRotate: 5,     // 画布旋转（秒）
  trickHints: true,    // 反向提示
  muteVoice: true,     // 只能画不能说
  antonym: true,       // 反义词挑战
  oppositeDraw: true,  // 相反画法
  abstractMode: true,  // 抽象大师
  wheel: true,         // 幸运轮盘
  boss: 10             // Boss 关（词数）
}
```

### 词库

- easy: 200 词（动物、水果、日用品、交通工具、运动用品）
- medium: 200 词（职业、地点、天气、动作）
- hard: 200 词（校园、网络、生活、影视）
- expert: 100 词（高辨识度事物）
- hard2: 100 词（情绪、人生、网络热词、关系）
- oneStroke: 100 词（简单可辨识事物）
- ink: 100 词（快速能画事物）
- antonym: 110 词（反义词对）
- oppositeDraw: 106 词（相反画法词对）
- trickHint: 106 词（反向提示词库）
- boss: 50 词（Boss 关混合词库）
- infinite: 96 词（无限模式专属）

**总计约 1368 词**，跨关卡去重，同一房间内不重复。

### 笔画同步

每个笔画事件包含：
- `x`, `y`: 坐标（含抖手偏移）
- `color`: 颜色
- `width`: 宽度（含笔刷动画）
- `mirror`: 镜像标志

服务端透传所有字段，客户端根据标志绘制对应效果。

### 隐形墨水

每个笔画记录时间戳，客户端通过 `requestAnimationFrame` 循环重绘，根据时间差计算透明度实现淡出效果。

### 画布旋转

旋转 `canvas-wrapper` 的 CSS `transform`，同时在 `_getPos` 中对触摸坐标做反向旋转变换，确保触摸位置与画面对应。
