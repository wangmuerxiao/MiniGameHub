# 数独游戏

独立的数独游戏项目，不影响五子棋内容。

## 启动方式

```bash
cd D:\个人研究\网站\小游戏\shudu\server
node server.js
```

然后访问 http://localhost:3001

## 功能

### 单人模式
- 简单难度：30-40 个给定数字
- 中等难度：25-29 个给定数字
- 困难难度：20-24 个给定数字

### 提示功能
- 每局有 3 次提示机会
- 点击提示按钮会随机填入一个正确数字

### 操作说明
- 点击格子选中
- 使用数字键盘或键盘 1-9 输入数字
- 使用退格键或删除键擦除数字
- 使用方向键移动选中的格子

### 其他功能
- 计时器
- 错误计数
- 撤销操作
- 检查解答
- 重新开始

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- 后端：Node.js
- 数独生成：qqwing 库
